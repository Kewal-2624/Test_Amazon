# Test_amazon
> This test case is used to fetch some deatil from Amazon.com 

## Build Setup
```bash
#install Selenium
pip install selenium

#install PyTest
pip install pytest

#install Pytest-html
pip install pytest-html


```

## Build Setup
```bash
# To run project
pytest -s

# To get short test summary info
pytest -s -rA

# To generate HTML report
pytest -s -rA --html=Report.html
```
